# üí† Ubuntu Compgen üõÖ

**About It** ‚û§ It is a type of bash builtin commands found in **Ubuntu**,**MACOS**, **BSD** and Operating System's those which are similar to UNIX Systems. ‚¶ø It is generally used in order to get displayed the available possible completions depending upon the type of arguments given.

‚û± Let's have a look on its command's format together by using optional arguments in its syntax.

